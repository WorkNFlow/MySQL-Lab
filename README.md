## 📌 Описание задания
1. Научиться работать с базой данных MySQL через PHP.
2. Создать таблицу для хранения данных формы.
3. Сохранять данные формы в базу данных.
4. Выводить сохранённые данные на странице.
5. Использовать классы PHP для работы с таблицей.
6. Научиться работать с Docker-контейнерами: nginx, PHP-FPM, MySQL, Adminer.

Вариант: покупка билета в кино (имя, количество билетов, фильм, чекбокс «3D очки», радио «Тип места»).

Результат доступен по адресу [http://localhost:8080](http://localhost:8080).

---

## ⚙️ Как запустить проект

1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/WorkNFlow/MySQL-Lab.git
   cd MySQL-Lab
   ```
2. Запустить контейнеры:
   ```bash
   docker-compose up -d --build
   ```
3. Открыть в браузере:
   ```text
   http://localhost:8080
   ```
4. Открыть Adminer для работы с базой данных:
   ```text
   http://localhost:8081
   ```

### Параметры подключения к БД

- Хост для PHP-контейнера: `db`
- Хост для локального клиента: `127.0.0.1`
- Порт: `3307`
- База данных: `lab5_db`
- Пользователь: `lab5_user`
- Пароль: `lab5_pass`

## 📂 Содержимое проекта

```text
docker-compose.yml  — описание сервисов nginx, php, db, adminer
Dockerfile          — образ PHP-FPM с поддержкой pdo и pdo_mysql
nginx/default.conf  — конфигурация nginx
www/
  index.php         — вывод всех сохранённых покупок билетов
  form.html         — форма покупки билета в кино
  process.php       — обработка формы и сохранение данных в БД
  db.php            — подключение к MySQL и создание таблицы students
  Student.php       — класс для работы с таблицей students
```
